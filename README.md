## installation

 1. Install this repo.
 2. Create a new folder and include a file `startSite.js` with the following code:
 ```javascript
 var wb = require('webBuilder');
 var options = {port:'8080'}
 wb.start(options);
 ```
 3. Run the file and navigate in your browser to: `localhost:8080`
